Prefix sum - Pattern 2

Prefix sum means, at every index, store the sum of all elements before it(including itself)
if array is:
[2,4,1,3]
Prefix sum becomes:
[2,6,7,10]

Meaning :
prefix[0]=2
prefix[1] =6
prefix[2]=7
prefix[3]=10

Why do we use prefix sum?
It allows you to answer what is the sum between L and R

in O(1) time complexity - Instead of summing every time in O(n)

WHen do we use Prefix Sum?
-> Sum for L and R
-> Range sum
-> Sub array sum equals K
-> Count subarrays
-> Multiple queries
-> Cumulative sum
-> prefix
